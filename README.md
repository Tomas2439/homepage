## 🎨 Concepto de diseño

El portfolio rompe con la estructura clásica de los portfolios de desarrolladores. En lugar del típico diseño flat con cards sólidas, implementa una estética **Liquid Glass** inspirada en visionOS de Apple:

- **Paneles translúcidos** con `backdrop-filter: blur()` real
- **Aurora borealis animada** como fondo vivo con gradientes en movimiento
- **Cards con tilt 3D** que responden al movimiento del mouse
- **Scroll reveal** escalonado en cada sección

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|---|---|
| **HTML5** semántico | Estructura y accesibilidad |
| **CSS3 Vanilla** | Sistema de diseño completo (variables, animaciones, grid) |
| **JavaScript Vanilla** | Todas las interacciones sin frameworks |
| **Tailwind CSS v4** (CDN) | Clases utilitarias de apoyo |
| **GSAP + ScrollTrigger** (CDN) | Animaciones de scroll |
| **Devicon** (CDN) | Iconos de tecnologías |
| **Google Fonts** | Outfit + Space Grotesk |

## ⚡ Features

- 🌌 Aurora borealis con orbs animados (optimizada para bajo consumo de GPU)
- 🫧 Glassmorphism en todos los paneles y navegación
- 🖱️ Cursor glow que sigue al mouse (throttled a 30fps)
- ✨ Cards con efecto tilt 3D + reflejo de luz en hover
- 📊 Contadores animados con easing cúbico
- 🔢 FPS Monitor: activa automáticamente un modo de bajo rendimiento si detecta lag
- 📱 Totalmente responsive (320px → 1920px)
- ♿ Accesible: roles ARIA, focus-visible, `prefers-reduced-motion`
- 🚀 Sin build steps, sin npm, sin dependencias locales
