Cómo desarrollo
con IA

Integro agentes de inteligencia artificial en cada etapa del desarrollo — no como reemplazo del pensamiento, sino como amplificador. Saber qué pedirle a cada herramienta y cómo evaluar su output es una skill en sí misma.

Antigravity (AGY)

Agente de código principal

Mi agente de desarrollo principal. Lo uso para planificar arquitecturas, refactorizar código, depurar errores complejos y construir features completas. Trabaja directamente sobre el filesystem.

Planificación Refactoring Debug Full features

Claude (Anthropic)

Razonamiento y análisis

Ideal para entender conceptos complejos en profundidad, revisar lógica de algoritmos y obtener explicaciones detalladas. Excelente para aprender el "por qué" detrás del código.

Conceptos Algoritmos Explicaciones

Gemini (Google)

Brainstorming y contenido

Uso Gemini para consultas rápidas, generar ideas de diseño, crear contenido de texto y explorar soluciones creativas gracias a su integración con el ecosistema de Google.

Brainstorming Consultas Contenido

Mi workflow con IA

Cómo los integro

La IA no escribe el código por mí — define el plan, yo apruebo. Reviso cada output, entiendo lo que hace y lo adapto. El objetivo es aprender más rápido, no depender de la IA para pensar.
